275 H-Index II
=======================
[Problem description](https://leetcode.com/problems/h-index-ii/)

#### Difficulty
<span style="color:#FABC60">Medium</span>

#### Keywords
- Binary search

#### Idea

#### Complexity
- Runtime: O(logN)
- Space: O(1) 
  
#### LC performance
- Runtime: 44 ms
- Memory usage: 16.6 MB

#### Code
```python
class Solution:
    def hIndex(self, citations: List[int]) -> int:
        if not citations:
            return 0
        lo, hi = 0, len(citations)
        while lo < hi:
            mid = (lo + hi)//2
            v = citations[mid]
            n = len(citations) - mid
            if n == v:
                return v
            if n > v:
                lo = mid + 1
            else:
                hi = mid
        return len(citations) - lo
```
