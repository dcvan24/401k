116 Populating Next Right Pointers in Each Node
=======================
[Problem description](https://leetcode.com/problems/populating-next-right-pointers-in-each-node/)

#### Difficulty
<span style="color:#FABC60">Medium</span>

#### Keywords
- Breadth-first search

#### Idea


#### Complexity
- Runtime: O(N)
- Space: O(1)

#### LC performance
- Runtime: 48 ms
- Memory usage: 15.2 MB

#### Code
```python
"""
# Definition for a Node.
class Node:
    def __init__(self, val, left, right, next):
        self.val = val
        self.left = left
        self.right = right
        self.next = next
"""
class Solution:
    def connect(self, root: 'Node') -> 'Node':
        from collections import deque
        
        q = deque([(root, 0)])
        p, cur_lv = None, -1
        while q:
            n, lv = q.popleft()
            if not n:
                continue
            if lv > cur_lv:
                cur_lv = lv
                p = n
            else:
                p.next = n
                p = p.next
            lv += 1
            if n.left:
                q += (n.left, lv),
            if n.right:
                q += (n.right, lv),
        
        return root
```
