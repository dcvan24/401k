235 Decode String
=======================
[Problem description](https://leetcode.com/problems/decode-string/)

#### Idea


#### Complexity
- Runtime: O(N)
- Space: O(N)
  
#### LC performance
- Runtime: 32 ms

#### Code
```python
class Solution:
    def decodeString(self, s: str) -> str:
        stack = [[1, []]]
        num = 0
        for c in s:
            if c.isnumeric():
                num = num * 10 + int(c)
            elif c == '[':
                stack += [num, []],
                num = 0
            elif c == ']':
                cnt, par = stack.pop()
                stack[-1][1] += ''.join(par) * cnt,
            else:
                stack[-1][1] += c,
        return ''.join(stack[-1][1])
```
