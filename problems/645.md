645 Set Mismatch
=======================
[Problem description](https://leetcode.com/problems/set-mismatch/)

#### Difficulty
<span style="color:green">Easy</span>

#### Keywords
- Hash indexing

#### Idea

#### Complexity
- Runtime: O(N)
- Space: O(N)

#### LC performance
- Runtime: 80 ms
- Memory usage: 14.7 MB

#### Code
```python
class Solution:
    def findErrorNums(self, nums: List[int]) -> List[int]:
        occurs, dup = set(), -1
        for i, n in enumerate(nums):
            if n in occurs:
                dup = n
            else:
                occurs.add(n)
        for i in range(1, len(nums) + 1):
            if i not in occurs:
                return [dup, i]
        return [dup, len(nums)]
```
