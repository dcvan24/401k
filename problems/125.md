125 Valid Palindrome
=======================

#### Idea

#### Complexity

#### LC performance

#### Code
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        s = [c.lower() for c in s if c.isalnum()]
        if not s:
            return True 
        i, j = 0, len(s) - 1
        while i < j:
            if s[i] != s[j]:
                return False 
            i += 1
            j -= 1
        return True 
```
