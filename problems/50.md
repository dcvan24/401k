50 Pow(x, n)

#### Idea

#### Complexity
- Runtime: O(logN)
- Space: O(1)

#### LC performance

#### Code
```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if x == 0:
            return 0
        if n < 0:
            x = 1/x
            n = -n
        res = 1
        prod, multi = x, 1
        while n > 0:
            res *= prod
            n -= multi
            multi += multi
            prod *= prod
            if multi > n:
                prod, multi = x, 1                
        return res
```
