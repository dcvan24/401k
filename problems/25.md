25 Hamming Distance
=======================
[Problem description](https://leetcode.com/problems/hamming-distance/)

#### Difficulty
<span style="color:green">Easy</span>

#### Keywords
- Bit operations

#### Idea

#### Complexity
- Runtime: O(N)
- Space: O(1)

#### LC performance
- Runtime: 36 ms
- Memory usage: 13.3 MB

#### Code
```python
class Solution:
    def hammingDistance(self, x: int, y: int) -> int:
        dist = 0
        while x > 0 or y > 0:
            xb, yb = x%2, y%2
            if xb != yb:
                dist += 1
            x >>= 1
            y >>= 1
        return dist
```
