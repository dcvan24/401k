985 Sum of Even Numbers After Queries
=======================
[Problem description](https://leetcode.com/problems/sum-of-even-numbers-after-queries/)

#### Difficulty
<span style="color:green">Easy</span>

#### Keywords
- Basic arithmetics 
- Array

#### Idea

#### Complexity
- Runtime: O(N)
- Space: O(1)

#### LC performance
- Runtime: 180 ms
- Memory usage: 17.6 MB

#### Code
```python
class Solution:
    def sumEvenAfterQueries(self, A: List[int], queries: List[List[int]]) -> List[int]:
            s, res = 0, []
            for n in A:
                if n%2 == 0:
                    s += n
            for v, i in queries:
                was_even = A[i]%2 == 0
                is_even = (A[i] + v)%2 == 0
                if was_even and is_even:
                    s += v
                elif was_even:
                    s -= A[i]
                elif is_even:
                    s += A[i] + v
                A[i] += v
                res += s,
            return res
```
