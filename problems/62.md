62 Unique Paths
=======================
[Problem description](https://leetcode.com/problems/unique-paths/)

#### Difficulty
<span style="color:#FABC60">Medium</span>

#### Keywords
- Dynamic programming
  
#### Idea


#### Complexity
- Runtime: O($N^2$)
- Space: O($N^2$)
  
#### LC performance
- Runtime: 40 ms
- Memory usage: 13.2 MB

#### Code
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[0] * n for _ in range(m)]
        for i in range(m):
            for j in range(n):
                if i == 0 or j == 0:
                    dp[i][j] = 1
                else:
                    dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
        return dp[-1][-1]
```
